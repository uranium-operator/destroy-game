# destroy-game
A game where you can destroy a School, the Statue of Liberty, etc
The object's HP decreases by 1 when typed by the user
Used a simple GUI in a batch language

# Feature (Lastest Version)
-GUI v1.0

-HP

-Achievement

-Save/Load System

# Command (Lastest Version)
-help

-save

-load

-quit

# How To Play
1. Download the lastest game
2. Double click the exe file
4. click "ok" if you done reading the message box
5. type your object into input box (must same as the list)
6. click "ok" and your object HP was decreased by 1
